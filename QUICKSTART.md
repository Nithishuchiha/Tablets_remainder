# Quick Start Guide

## Installation

1. **Install Backend**:
```bash
cd backend
npm install
```

2. **Install Frontend**:
```bash
cd frontend
npm install
```

## Running the App

### Terminal 1 - Backend:
```bash
cd backend
npm start
```
**Scan the QR code with WhatsApp!**

### Terminal 2 - Frontend:
```bash
cd frontend
npm start
```

## First Time Setup

1. **Scan QR Code**: When backend starts, scan the QR code in terminal with WhatsApp
2. **Add Contacts**: Go to Contacts tab, add family members with phone numbers (format: country code + number)
3. **Add Medications**: Go to Medications tab, add medicines with timings
4. **Test**: Use the "Test" button on contacts to verify WhatsApp is working

## Phone Format Examples
- India: 919876543210
- USA: 1234567890
- UK: 441234567890
